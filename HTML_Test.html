1. Which of the following is used to create Web Page ?

  1. HTML
  2. JVM
  3. DTD
  4. C
Answer: HTML

2. HTML Stands for _____________ ?

  1. Hyper Text Makeup Language
  2. Hyper Text Markup Language
  3. None of these
  4. Hyper Tech Markup Language
Answer: Hyper Text Markup Language

3. HTML is considered as _________ language ?

  1. Higher Level Language
  2. Markup Language
  3. OOP Language
  4. Programming Language
Answer: Markup Language

4. HTML markup language is a set of Markup ___________ ?

  1. Tags
  2. Attributes
  3. Groups
  4. Sets
Answer: Tags

5. HTML tags are used to describe document ___________ ?

  1. None of these
  2. Content
  3. Definition
  4. Language
Answer: Content
